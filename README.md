# WayPilgrim (순례자의 길)
삶 속에서의 예배를 돕는 육각형 엔진 기반 신앙 공동체 플랫폼입니다.

## 6대 엔진 아키텍처
1. **동행 (Companionship)**: 자발적 소통 광장
2. **지혜의 보고 (Library)**: 말씀과 고백의 성소
3. **성벽 (Rampart)**: 보안 및 영적 보호
4. **미관 (Aesthetic)**: 영성적 UX/UI
5. **흐르는 시내 (Overflow)**: 삶의 실천 및 전파
6. **청지기 (Steward)**: [Core] 통합 관리 및 AI 비서 (GitHub 연동)
